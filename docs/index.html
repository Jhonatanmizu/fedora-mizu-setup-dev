🐧 Fedora Mizu Setup

A minimal, fast, and elegant script to set up your Fedora-based development environment.

🚀 Getting Started

Clone the repo and run the script:

git clone https://github.com/Jhonatanmizu/fedora-mizu-setup-dev.git
cd fedora-mizu-setup-dev
chmod +x setup.sh
./setup.sh

⚙️ What it Does

  • 📦 Installs essential packages and tools
  • 🧩 Configures GNOME extensions, themes, and keyboard shortcuts
  • 🛠️ Applies dotfiles using stow
  • 🎨 Uses the Tokyo Night-inspired visual setup
